# Task 8 - Mini Project Planning

## Project Name
Customer Churn Prediction System

## Dataset
E-commerce Customer Churn Dataset

## Project Objective
To predict whether a customer is likely to churn based on customer demographics, purchasing behavior, and engagement history.

## Features (X)
- age
- tenure_months
- avg_order_value
- total_orders
- last_purchase_days_ago
- support_tickets

## Target Variable (y)
- churn

## Machine Learning Model
- Logistic Regression

## Business Question
Can we predict whether a customer will leave the e-commerce platform using historical customer data?

## Expected Outcome
The model will identify customers who are at risk of churning, enabling the business to take proactive retention measures and improve customer satisfaction.

## Conclusion
This project aims to leverage machine learning techniques to predict customer churn and support data-driven decision-making for customer retention strategies.
