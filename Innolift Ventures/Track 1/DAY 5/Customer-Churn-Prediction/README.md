# Customer Churn Prediction System - E-commerce

## Project Overview
This project predicts whether an e-commerce customer is likely to churn based on customer demographics, purchase history, and engagement metrics.

## Problem Statement
Customer churn leads to revenue loss for e-commerce businesses. The goal of this project is to identify customers who are likely to stop using the platform so that retention strategies can be applied.

## Dataset
E-commerce Customer Churn Dataset

## Features Used
- Age
- Gender
- City
- Tenure Months
- Average Order Value
- Total Orders
- Last Purchase Days Ago
- Support Tickets
- Subscription Type

## Target Variable
- Churn (0 = Stay, 1 = Churn)

## Algorithm Used
- Logistic Regression

## Model Performance
- Accuracy: 65.55%

## Project Files
- model.py
- predict.py
- charts.py
- model.pkl
- churn_distribution.png
- feature_vs_churn.png
- correlation_heatmap.png

## How to Run

1. Train the model:
   python model.py

2. Run predictions:
   python predict.py

3. Generate charts:
   python charts.py

## Output
The model predicts whether a customer is likely to stay or churn and generates visualizations for analysis.
