About Student Registration Portal

The Student Registration Portal is a web-based application developed to manage student information efficiently. It allows administrators to register students, store their academic details, and organize records department-wise.


This project is built using Flask, Python, HTML, CSS, SQLite, and Jinja Templates. The system provides an easy-to-use interface for maintaining student records and accessing information quickly.


Key Features:

Student Registration

Department-wise Student Records

Student Database Management

Delete Student Records

Responsive Glassmorphism User Interface
